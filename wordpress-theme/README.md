# Dependable Marketing Solutions - WordPress Theme

A modern, standalone WordPress theme converted from React.

## Installation Steps

1.  **Upload Theme**:
    * Compress the `wordpress-theme` folder into a `.zip` file.
    * Go to WordPress Admin > Appearance > Themes > Add New > Upload Theme.
    * Upload and activate.

2.  **Install Plugins**:
    * **Advanced Custom Fields (ACF)**: Required to manage the Team and Services sections.

3.  **Setup Pages**:
    * Create a page named "Home" -> Assign Template: **Front Page**
    * Create a page named "About" -> Assign Template: **About Page**
    * Create a page named "Services" -> Assign Template: **Services Page**
    * Create a page named "Contact" -> Assign Template: **Contact Page**
    * Go to Settings > Reading -> Set "Your homepage displays" to "A static page" and select "Home".

4.  **Sync Settings**:
    * Go to Custom Fields > Tools > Sync (if available) OR check the "Theme Settings" menu item that appears in the sidebar to populate your content.

5.  **Menus**:
    * Go to Appearance > Menus.
    * Create a "Primary Navigation" menu and assign it to the "Primary Navigation" location.
